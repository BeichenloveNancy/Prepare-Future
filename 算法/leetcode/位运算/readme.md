# 按位操作
  - 按位与
    每一位都是1，结果才为1

    8 & 7
    1000
     &
    0111 => 0000

  - 按位或
    只要1位是1 结果就为1
    
    8 | 7 => 15
 
  - 按位异或
    每一位都不同，结果才是1

    8 ^ 7 => 15
